# HelloApp Use Cases

## UC1 - Display Hello World
The application prints "Hello World" on the screen.

## UC2 - Display User Name
The application asks the user for their name and displays it.

## UC3 - Accept Name from Command Line
The application accepts the user name from command line arguments.

## UC4 - Store Names in Collection
The application stores multiple names using a list.

## UC5 - Remove Name
The application allows removing a name from the list.

## UC6 - Save Names
The application saves the names for future runs.

## UC7 - Load Names
The application loads previously saved names.

## UC8 - Display Banner
The application displays the names in a banner format.
